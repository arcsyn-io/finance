# Plano: Painel patrimonial e carregamento de autenticação

1. Registrar o comportamento de carregamento estável e as dimensões financeiras do painel.
2. Criar teste de serviço para consolidar a posição patrimonial a partir de um contrato de repositório.
3. Implementar repositório, command, DTO, service e factory do painel.
4. Mapear o DTO para uma view model e compor a nova tela inicial.
5. Substituir o controle transitório de login e MFA por estado de requisição assíncrona persistente.
6. Validar testes, lint, tipos, build e a interface em navegador.
